# kiwijs.github.io

## 目录结构

```
css/              存放所有的css文件
----image/        存放所有的网页图片
design/           存放所有的设计稿
----png/          切好的图标、背景之类
----psd/          PSD文件
js/               存放所有的js文件
index.html        首页
```

## 安装和启动服务器

安装（若没有安装过nodejs，请点击此处[下载](https://nodejs.org/dist/v4.5.0/node-v4.5.0.pkg)安装包）
```
npm i
```

启动
```
npm run dev
```

然后在浏览器中打开 http://localhost:3030/
